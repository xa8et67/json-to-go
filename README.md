# json2go

## Introduction

将json直接转为go结构体，支持备注，自定义tag，中文属性，属性类型推断，属性合并等功能，并提供简单易用的web界面

## Web

在线地址(chrome): https://zzlgo.github.io/json2go

## Features

* 支持单行备注，可在上一行或行尾
* 支持自定义tag
* 支持属性名格式化，支持中文属性
* 支持属性类型推断，单独属性，数组内属性，同名对象属性
* 支持对象属性合并，重名的所有对象，包括数组内所有对象
* 基于wasm，提供简单易用的web界面
